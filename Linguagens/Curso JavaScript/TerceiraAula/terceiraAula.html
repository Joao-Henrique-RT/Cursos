Tipos primitivos


Existem dois tipos primitivos, de referência e valores

Referência: String, number, boolean, null, undefined, Symbol


String, é um conjunto de letras que formam uma palavra ou frase

Boolean, estrutura lógica do sistema, ex: aprovado ou não, true e false

Null, serve para resetar alguma informação, ex: quando uma pessoa escolhe a cor de uma página
